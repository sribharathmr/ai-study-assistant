# AI Study Assistant (RAG Chatbot)

An offline AI assistant that answers questions from study notes.

## Features
- Semantic search using embeddings
- Retrieval Augmented Generation (RAG)
- Local LLM using Ollama (phi3)
- Works without internet

## Tech Stack
Python, SentenceTransformers, Ollama, NLP, LLM

## How it works
Question → Retrieve relevant notes → AI explains
